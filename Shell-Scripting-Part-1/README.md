# Shell Scripting for DevOps - Part 01

## Introduction

Started Shell Scripting as part of my DevOps Zero to Hero journey.  
This part focused on understanding Linux basics, file handling, permissions, shell concepts, and practical scripting fundamentals.

- Writing first simple shell script

### 7. Permissions & Security
- chmod command

- File execution permissions

### 8. Command History
- history command
- Revising previously used commands

### 9. System Monitoring Basics
- Checking CPU and RAM
- top command
- Basic system monitoring
