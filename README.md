# DevOps Mastery Journey

This repository documents my structured learning and hands-on practice in DevOps and Cloud Engineering.

The goal of this project is to build strong fundamentals, gain real-world practical exposure, and develop the skills required to work as a DevOps / Cloud Engineer.


## Application Source Code

This repository primarily documents the DevOps workflow used to deploy a Node.js application on AWS EC2, including infrastructure setup, deployment steps, troubleshooting, and key concepts learned during the process.

The full application source code used in this project is maintained in a separate forked repository.

The forked repository contains the main implementation files of the application, including frontend resources, server files, and project configuration.

You can explore the complete application code here:

https://github.com/krishnakala987-byte/Deploy-NodeJS-on-AWS.git

The forked repository includes files and directories such as:

* client/
* assets/
* css/
* workshops/
* Speaker_Imgs/
* app.js
* index.html
* cancel.html
* success.html
* server.js
* package.json
* Dockerfile
* static assets and images

These files represent the main application logic and frontend resources, while this repository focuses on documenting the DevOps deployment process and infrastructure setup.
