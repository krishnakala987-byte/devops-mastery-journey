This section of my DevOps learning journey focuses on understanding Git and GitHub, which are essential tools for version control and collaboration in modern software development.

## Topics Covered

- Introduction to Version Control
- Why Version Control is important in software development
- Centralized vs Distributed Version Control Systems
- Introduction to Git
- Introduction to GitHub
- Installing and configuring Git
- Initializing a Git repository
- Understanding the `.git` directory
- Working with files in a Git repository
- Staging changes using `git add`
- Creating versions using `git commit`
- Writing meaningful commit messages
- Viewing commit history
- Understanding commit IDs and version history
- Navigating to previous versions
- Understanding Git workflow

- Observed how Git tracks changes
- Explored the contents of the `.git` directory
- Learned how Git stores versions internally
- Practiced navigating commit history
- Understood how to move to previous versions of code
